Passo Um - Características Pessoais.

Motivação.

Com este estudo, pretendemos:

- Entender que misturas de cores são facilmente percebidas pelo sistema visual humano.

- Perceber se a mistura de cores é um bom canal para mostrar informação.

- Concluir se uma pessoa consegue tanto construir mentalmente misturas de cor, como desconstruir misturas nos seus componentes básicos.
